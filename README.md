## Overview
This project is the first case study from the 
[Google Data Analytics Professional Certificate](https://www.coursera.org/professional-certificates/google-data-analytics) program. The analysis focuses on Cyclistic, a fictional bike-share company based in Chicago, and explores how different customer groups use the service.

The primary business objective of this project is to understand the behavioral differences between casual riders and annual members in order to support Cyclistic’s long-term membership growth strategy. Since annual members are more profitable for the company, the marketing team aims to develop data-driven strategies that can convert casual riders into loyal members.

Using SQL and R as the main analytical tools, this project follows the complete data analysis process including data cleaning, transformation, exploratory analysis, visualization, and business recommendation development.

The analysis examines ride behavior patterns such as:
- Ride frequency
- Ride duration
- Weekly usage trends
- Differences between member and casual riders

The final result of this project includes insights, visualizations, and strategic recommendations designed to help Cyclistic improve customer conversion and support data-informed marketing decisions.

## Business Task
Cyclistic’s marketing team wants to increase the number of annual members because they are more profitable than casual riders. 

The objective of this project is to identify behavioral differences between casual riders and annual members and recommend marketing strategies that encourage membership conversion.

## Data Source
The dataset used in this project contains historical trip data from Cyclistic bike-share users.

Data includes:
- Ride duration
- Start & end stations
- Ride dates
- Rider type (member or casual)

---

## Data Cleaning & Processing
The following steps were completed during the cleaning process:

- Removed duplicates and null values
- Standardized column formats
- Created calculated fields:
  - `ride_length`
  - `day_of_week`
- Combined datasets into a single analysis file
- Filtered invalid ride durations

---

## Key Questions
1. How do annual members and casual riders use bikes differently?
2. What patterns exist in ride duration and ride frequency?
3. Which days are most active for each rider type?
4. How can marketing strategies target casual riders effectively?

---

## Key Insights
- Casual riders tend to take longer rides
- Annual members ride more consistently throughout the week
- Casual riders are more active on weekends
- Members are more likely to use bikes for commuting purposes

---

## Recommendations
### 1. Weekend Membership Campaigns
Target casual riders during weekends when usage is highest.

### 2. Digital Marketing Strategy
Use social media campaigns and app notifications to promote annual memberships.

### 3. Incentive Programs
Offer discounts or rewards for frequent casual riders to encourage membership upgrades.
